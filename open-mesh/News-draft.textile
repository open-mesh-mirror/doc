h1. DRAFT: Batman-adv 2025.5 released

*December 16nd, 2025*: The B.A.T.M.A.N. team is excited to announce the release of the December 2025 update for batman-adv! This release includes various code cleanups and drops support for EOL kernels.

Since the kernel module is always dependent on the specific Linux kernel itâ€™s compiled against, providing precompiled binaries on our website is not feasible. As always, you can find the signed tarballs in the download section:

https://downloads.open-mesh.org/batman/releases/batman-adv-2025.5/
